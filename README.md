# Bi Games

Site oficial da loja **Bi Games**, localizada em Itaquaquecetuba (Av. Italo Adami, 396B).

## Sobre o Projeto
Este projeto é uma plataforma de e-commerce desenvolvida para venda e conserto de consoles, jogos e acessórios. O objetivo é permitir que os clientes comprem online e retirem os produtos diretamente na loja física, similar ao modelo de operação da *Action Game*, mas com identidade visual e funcionalidades exclusivas da Bi Games.

## Funcionalidades Principais
- **Catálogo de Produtos:** Consoles, Jogos, Acessórios, Geek e Informática.
- **Sistema de Vendas:** Carrinho de compras e checkout para retirada na loja.
- **Serviços:** Solicitação/informações sobre conserto de consoles.
- **Integração:** Links diretos para WhatsApp e localização.

## Identidade Visual
- **Cores:** Vermelho, Branco e tons de Cinza/Beige.
- **Conceito:** Temática gamer com personagens clássicos (estilo Mario/Yoshi) e tipografia moderna/techno.
